## 🚨 THE 4 CRITICAL ISSUES (Must Fix First)

1. **PDF Export Fails in Production**
   - Location: `apps/web/app/api/export-pdf/route.ts`
   - Problem: Puppeteer crashes in Vercel (no Chrome binary)
   - Fix: Add @sparticuz/chromium
   - Time: 2-3 hours
   - Status: 🔴 CRITICAL

2. **Database Query Returns Duplicate Data**
   - Location: `apps/web/app/server/db/queries/question-papers.ts`
   - Problem: Cartesian product from wrong JOIN
   - Fix: Add chapterId field + fix JOIN
   - Time: 3-4 hours
   - Status: 🔴 CRITICAL

3. **Hardcoded Subject/User IDs**
   - Location: `apps/web/app/api/question-papers/route.ts`
   - Problem: All papers saved to subject ID 1
   - Fix: Add subject selection UI + remove fallbacks
   - Time: 4-6 hours
   - Status: 🔴 CRITICAL

4. **Weak ID Generation (Math.random)**
   - Location: Multiple API routes
   - Problem: High collision risk in production
   - Fix: Use UUID or database auto-increment
   - Time: 1-2 hours
   - Status: 🔴 CRITICAL
