# Question Hive - Future Scope & Feature Roadmap

## 🎯 Project Overview
Question Hive is a SaaS platform designed for professors and educational institutions to create high-quality competitive exam question papers (JEE, NEET, CET) using AI generation and existing question randomization with an intuitive drag-and-drop PDF builder.

## 📋 Phase 1 - MVP Features (Current Focus)
### Core Functionality
- ✅ User authentication for professors
- ✅ Subject & chapter selection (11th/12th - PCM + Biology)
- ✅ AI question generation using Gemini API
- ✅ Question randomizer from database
- ✅ Drag & drop PDF builder with free-form placement
- ✅ Basic question paper generation (A4 format)
- ✅ Dual PDF export (Questions only + Questions with answers)

### UI/UX Components
- ✅ Split-screen interface (Questions left, PDF builder right)
- ✅ Question difficulty selection based on exam type
- ✅ Basic equation editor integration
- ✅ Watermark and header/footer customization

### Technical Foundation
- ✅ Go backend with Gin framework
- ✅ Next.js frontend
- ✅ PostgreSQL database
- ✅ Gemini AI integration
- ✅ File storage for PDFs and images
